Excel 1 Project Overview


The Excel Project 1 was designed to display my skills in basic Excel functions and tools. Here, data provided by my ISM3011 course was used to create a workbook that displays this data in different ways. The project consisted of different worksheets designed to present the summarized data for a specific title of music, pivot tables with specific goals, etc. Here is a list of the functions, tools, and excel features used in this project:
  • Data Validation
  • SUM, IF, VLOOKUP and other functions
  • Naming cell ranges and referencing them
  • Adding and designing different kinds of borders to cells and tables
  • Changing font properties and cell background colors
  • Displaying graphic representation of data in specific cells
  • Using pivot tables and charts
  • Sorting lists and using filters to display desired results
  • Protecting and locking cells in a sheet
  • Creating and renaming worksheets in the document
