Excel 2 Project Overview


Similar to Excel Project 1, this was designed to further display my skills in basic Excel functions and tools. Here, medical data for different states provided by my ISM3011 course was used to create a workbook that displays this data in different ways. The project consisted of different worksheets designed to present the summarized data for a specific state, pivot tables with specific goals, etc. Here is a list of the functions, tools, and excel features used in this project:
  • Data Validation
  • SUM, IF, VLOOKUP and other functions
  • Naming cell ranges and referencing them
  • Adding and designing different kinds of borders to cells and tables
  • Changing font properties and cell background colors
  • Displaying graphic representation of data in specific cells
  • Using pivot tables and charts with slicers
  • Sorting lists and using filters to display desired results
  • Protecting and locking cells in a sheet
  • Creating and renaming worksheets in the document

Different worksheets served different purposes. For example, the Comparisons worksheet showed the comparison of medical data for two different states, as entered by the user. This was also presented graphically as a pie chart. After this, the worksheets Pivot 1, 2, and 3 were created from the Medical Data sheet to display specific arithmetic values of particular fields in the data. Similarly, Filter 1 and Filter 2 are worksheets that display states with an above-average total policy score and states that limit charges to low-income and provide accurate itemized bills respectively.
